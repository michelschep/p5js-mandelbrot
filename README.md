## Over dit project

Deze applicatie visualiseert de beroemde Mandelbrot set - een fractale structuur die ontstaat door het itereren van een wiskundige formule. De prachtige patronen en kleuren komen voort uit de complexe dynamiek van eenvoudige wiskundige regels.

## Technologie

- [p5.js](https://p5js.org/) - JavaScript library voor creative coding
- HTML5 Canvas

## Lokaal uitvoeren

1. Clone de repository
2. Open `index.html` in een moderne webbrowser
3. Geniet van de fractals!
